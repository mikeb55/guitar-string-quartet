# Sylva Quartet — Project Notes

## Collaboration Context

This project is intended for collaboration with the Sylva Quartet. The music is written for real players, with future recording and performance in mind.

**Reference:** https://www.instagram.com/sylva.quartet/

## Priorities

- **Idiomatic string writing** — Playable, expressive, suited to quartet technique
- **Expressive parts** — Each part rewarding to play
- **Clear rehearsal logic** — Rehearsal letters, section markings, readable scores
- **Musical independence** — Voices can stand alone; not mere accompaniment
- **Playable but sophisticated** — Challenging without being impractical

## Recording Session

Mike Bryant’s first recording session with the Sylva Quartet established the collaborative foundation. This album represents the follow-up creative phase building on that experience.
