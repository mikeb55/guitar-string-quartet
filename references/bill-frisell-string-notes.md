# Bill Frisell String Writing — Notes

Lessons from Frisell’s string-oriented recordings (Richter 858, Sign of Life, Big Sur, When You Wish Upon a Star).

## Transparent Orchestration

- Every note audible
- No doubling for thickness
- One note per part when possible
- Sparse is default; dense is exception

## Minimal Density

- Fewer notes, more impact
- Rest and silence as material
- Avoid constant activity
- Full ensemble is an event, not a default

## Long Melodic Arcs

- Melodies unfold over many bars
- Avoid short, choppy phrases
- Lyrical lines with breath
- Guitar and strings share long phrases

## Chamber Interplay

- Instruments respond to each other
- Call and response
- Not accompaniment; conversation
- Each part has agency

## Guitar Integrated Rather Than Dominant

- Guitar weaves in and out
- Can drop out for string-only sections
- Can support when strings lead
- Equal partner, not soloist with backing

## Application to This Project

Use these principles when:
- Choosing texture (subset vs full ensemble)
- Writing melody (long arcs, breath)
- Orchestrating (transparency, no doubling)
- Placing guitar (integrated, not dominant)
