# Style Guide

Musical identity for this project.

## What This Music Is

- Chamber jazz with guitar and string quartet
- Spacious, transparent, lyrical
- Melody-driven harmony
- Independent voice movement
- Conversational ensemble interaction
- Asymmetrical phrasing
- Restraint and silence as expressive tools

## What This Music Is Not

- Generic cinematic string pads
- Conservative classical quartet writing
- Constant block-chord harmony
- Symmetrical algorithmic phrasing
- Every instrument playing every bar
- Dense, wall-of-sound texture

## Texture Priorities

1. **Transparency** — Every note audible; avoid doubling for thickness.
2. **Register separation** — Strings in distinct ranges; guitar in its idiomatic register.
3. **Subset writing** — Frequent use of 2–3 instruments; full quintet is an event.
4. **Chamber interplay** — Parts respond to each other; not tutti accompaniment.

## Voice-Leading Priorities

1. **Melodic independence** — Each voice has its own contour.
2. **Contrapuntal awareness** — Parallel motion used sparingly and intentionally.
3. **Non-root support** — Upper structures, suspensions, planing when appropriate.
4. **Smooth bass motion** — Cello anchors; stepwise when possible.

## Rhythmic Behaviour

1. **Asymmetrical phrase lengths** — 3, 5, 7 bars; avoid constant 4+4.
2. **Breath between gestures** — Rests; no continuous density.
3. **Phrase spillover** — Melodic lines can cross bar lines irregularly.
4. **Elasticity** — Notation allows rubato; avoid mechanical grid.

## Dynamic Behaviour

1. **Restraint** — *mp* and *mf* as default; *ff* rare.
2. **Gradient dynamics** — Crescendo and diminuendo over long spans.
3. **Per-instrument control** — Different dynamics per part when appropriate.
4. **Silence as dynamic** — Rests contribute to shape.

## Anti-Cliché Rules

- No constant whole-note string pads
- No formulaic I–IV–V–I progressions without melodic justification
- No mechanical 4-bar phrase symmetry
- No tutti every bar
- No generic “emotional” crescendos
- No doubling melody in octaves unless texturally intentional
