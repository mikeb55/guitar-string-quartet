# Part Preparation Checklist

Use before rehearsal or recording with Sylva Quartet.

## Score

- [ ] Instrument names correct (Guitar, Violin I, Violin II, Viola, Cello)
- [ ] Tempo marks at start and at changes
- [ ] Rehearsal letters at section boundaries
- [ ] Chord symbols present when relevant
- [ ] Readable spacing (4–6 measures per system)
- [ ] Clean rhythmic notation (no overlaps, clear beams)

## Parts

- [ ] Each part extracted and proofed
- [ ] Page turns at logical points
- [ ] Cues for long rests (if applicable)
- [ ] Dynamic markings clear
- [ ] Rehearsal letters matching score

## Rehearsal Readiness

- [ ] MusicXML exported and tested in Sibelius
- [ ] PDF generated for distribution
- [ ] Arrangement plan available for reference
