# Wyble Arrangement Notes — Eviscerating Angels

## Source file used

`compositions/eviscerating-angels/musicxml/V5_Eviscerating_Angels_Final.musicxml`

## Reason for creating the arrangement

To translate the dark narrative into Wyble two-line counterpoint. Preserves tension and controlled instability while reimagining guitar as sparse dyadic voices.

## Overview of the contrapuntal guitar approach

- **Upper voice:** 3rd, 7th of Am (C, G); dark, tense.
- **Lower voice:** Root and 5th (A, E); bass contour.
- **5/4 time:** Preserved; "Slowly, dark and tense."
- **Oblique motion:** One voice sustains; tension through restraint.

## Rehearsal letter structure

- **A** — Intro / tension
- **B** — Development
- **C** — Climax
- **D** — Outro

## Performance suggestions

- **Tempo:** 76 bpm; dark, unhurried.
- **Dynamics:** pp opening; build to climax; return to pp.
- **Strings:** Primary tension; guitar supports sparse.
