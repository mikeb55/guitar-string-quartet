# Wyble Arrangement Notes — Home Engine

## Source file used

`compositions/home-engine/musicxml/V9_Home_Engine_Scofield.musicxml`

## Reason for creating the arrangement

To offer an alternate performance version that translates the Scofield–Holland groove material into Jimmy Wyble's two-line contrapuntal guitar language. The Wyble arrangement preserves the harmonic world (Dm9) and emotional function while reimagining the guitar role as independent melodic voices rather than comping.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Carries 3rd, 7th, 9th (F, C, E, A) in a melodic line; moves in contrary or oblique motion to the lower voice.
- **Lower voice:** Carries root and 5th (D, A); provides bass contour and harmonic anchor.
- **Dyadic harmony:** Two-note combinations imply the full chord; no stacked jazz voicings.
- **Rhythmic independence:** Voices may move on different beats; one voice holds while the other moves (oblique motion).
- **Avoid:** Dense comping, block chords, constant monophonic line.

## Rehearsal letter structure

- **A** — Intro / main theme (Dm9)
- **B** — Development / solo section
- **C** — Thematic return
- **D** — Zappa-influenced section (if present)
- **E** — Climax
- **A'** — Return
- **Coda** — Outro

Sections may be extended or repeated in performance. The arrangement is modular.

## Performance suggestions

- **Phrasing:** Allow the two guitar lines to breathe; avoid rushing the counterpoint.
- **Balance:** Upper voice slightly forward for melody; lower voice anchors.
- **Strings:** Support without crowding; sustained pads, cello pedal, occasional counterlines.
- **Tempo:** 104 bpm as in source; Wyble style benefits from clarity at this tempo.
- **Dynamics:** Use the full range; contrast pp sections with mf/f climaxes.
