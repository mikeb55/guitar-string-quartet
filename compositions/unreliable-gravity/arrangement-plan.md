# Unreliable Gravity — Arrangement Plan

## Instrument Roles

| Instrument | Role |
|------------|------|
| **Violin I** | Fragmented melodic narrative; high-register instability; harmonics at transitions |
| **Violin II** | Delayed motif echoes; upper-colour shadowing; off-axis answers |
| **Viola** | Active chromatic inner voice; harmonic destabiliser; contrary-motion tension |
| **Cello** | Slow gravity lines; unexpected pedal shifts; register leap before climax |
| **Guitar** | Sparse harmonic instigator; dyads, triads, partial chords; drives section C texture |

## Textural Plan

1. **Sparse duo** — Guitar + cello or violin + viola
2. **Contrapuntal quartet** — Four independent lines
3. **Harmonic suspension** — Sustained chord fields
4. **Ensemble cluster** — Dense friction (climax)
5. **Contrast section** — Reduced texture, guitar harmonics
6. **Transformed return** — Memory of opening motif in new register

## Climax Logic

~70% through piece (section D). Strings converge into harmonic friction. Violin I highest register. Cello register leap. Guitar sharp dyads. f/ff. Sustain briefly before collapse.

## Ending Logic

No tonal resolution. Cello sustained ambiguous pitch. Violin harmonics or soft high suspension. Guitar harmonic dyad fading. Calm but unresolved.
