# Unreliable Gravity

**Engine:** Andrew Hill Harmonic Engine  
**Instrumentation:** Guitar, Violin I, Violin II, Viola, Cello  
**Meter:** 7/4 (4+3 pulse)  
**Duration:** ~5–6 minutes  
**Tempo:** ♩ = 62

## Current Recommended Version

**V2** — `musicxml/V2_Unreliable_Gravity_7-4.musicxml`

## Piece Description

A slow, dark chamber piece where shifting non-functional harmonies and fragmentary motifs make the tonal centre feel like it keeps appearing and slipping away while guitar and strings circle it without resolving.

## Engine Description

Andrew Hill Harmonic Engine: unstable tonal gravity, roots by minor third or tritone, harmonic ambiguity, no ii–V–I, angular but lyrical interior writing.

## Structure

- **A** (1–12): Unstable emergence  
- **B** (13–24): Harmonic drift  
- **C** (25–38): Motivic accumulation (guitar drives)  
- **D** (39–42): Gravitational collapse (climax)  
- **E** (43–52): Suspended aftermath
