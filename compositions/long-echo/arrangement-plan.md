# Long Echo – Arrangement Plan

## Instrument Roles

| Instrument | Role |
|------------|------|
| Guitar | Primary colour source; dyads, open triads, harmonics, sparse chord fragments |
| Violin I | High suspended line, harmonics, fragile melodic emergence |
| Violin II | Mid-high colour support, delayed echoes, soft inner suspension |
| Viola | Inner-voice glue, stepwise harmonic motion, dark warmth |
| Cello | Long pedal tones, slow contrary motion, gravitational anchor |

## Dynamic Map

- **A:** pp throughout
- **B:** pp → p (long crescendo)
- **C:** p, one brief mf crest (bars 35–37)
- **D:** p → pp (long decrescendo)
- **E:** pp to end

## Ending Logic

- Pre-ending dissolution: m.63 vn2 out, m.64 vla out, m.65 gtr out
- m.65: suspended (vn1 + cello), motivic recall D–E–A
- m.66: final sonority — soft guitar dyad (D–A), violin I (E6), long cello (D2)
- No final cadence; suspended, slightly unfinished
- Extra space before silence
