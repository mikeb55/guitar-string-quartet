# Wyble Arrangement Notes — Long Echo

## Source file used

`compositions/long-echo/musicxml/V2_Long_Echo_Final.musicxml`

## Reason for creating the arrangement

To translate the epilogue character into Wyble two-line language. Preserves stillness, decay, and open endings; guitar becomes sparse dyads that imply harmony without filling space.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Sparse; 3rd, 5th of D (F#, A).
- **Lower voice:** Root and 5th (D, A); minimal; sustained.
- **Oblique motion:** One voice holds; maximum space.
- **52 bpm:** Very slow; epilogue character.

## Rehearsal letter structure

- **A** — Intro / main theme
- **B** — Development
- **C** — Thematic return
- **D** — Outro / decay

## Performance suggestions

- **Stillness:** Allow long rests; avoid filling.
- **pp dynamics:** Wyble lines should be whisper-quiet.
- **Open ending:** Preserve epilogue character; suspended close.
