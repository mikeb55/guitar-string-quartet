# Wyble Arrangement Notes — Shifting Frames

## Source file used

`compositions/shifting-frames/musicxml/V2_Shifting_Frames_Final.musicxml`

## Reason for creating the arrangement

To translate the rhythmic architecture into Wyble two-line counterpoint. Preserves rhythmic cell logic and independence of voices while reimagining guitar as two melodic lines rather than dyadic comping.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Rhythmic; 3rd, 7th of Cmaj7 (E, B).
- **Lower voice:** Root and 5th (C, G); rhythmic displacement.
- **Two-line clarity:** Each voice has independent rhythm; no block harmony.
- **104 bpm:** Medium-up; rhythmic.

## Rehearsal letter structure

- **A** — Main theme
- **B** — Development
- **C** — Thematic return
- **D** — Outro

## Performance suggestions

- **Rhythmic independence:** Two lines should feel like independent rhythmic cells.
- **Avoid block harmony:** Do not stack into chords.
- **Strings:** Support without crowding; occasional counterlines.
