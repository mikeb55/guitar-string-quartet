# Shifting Frames – Arrangement Plan

## Instrument Roles

| Instrument | Role |
|------------|------|
| Guitar | Introduces cell; percussive dyads, muted attacks, syncopated fragments |
| Violin I | Fast displaced versions of cell |
| Violin II | Echo patterns, rhythmic counterpoint |
| Viola | Inner rhythmic glue |
| Cello | 7-beat cycle; grounds rhythmic field |

## Texture Map

- Guitar + cello rhythmic duet
- String trio cross-rhythm
- Full quintet rhythmic lattice
- Sudden unisons that fragment
- Refresh every 8–12 measures

## Ending Logic

Instruments drop out gradually. Displaced fragments of original cell. Final isolated gesture. No tonal resolution.
