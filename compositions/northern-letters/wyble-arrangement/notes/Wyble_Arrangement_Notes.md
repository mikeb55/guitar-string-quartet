# Wyble Arrangement Notes — Northern Letters

## Source file used

`compositions/northern-letters/musicxml/V2_Northern_Letters_Final.musicxml`

## Reason for creating the arrangement

To translate the Metheny/Bacharach lyrical material into Wyble two-line counterpoint. Protects the tune while reimagining the guitar role as contrapuntal rather than melodic lead.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Carries 3rd, 7th of Cmaj7 (E, B) in lyrical line.
- **Lower voice:** Root and 5th (C, G); warm bass contour.
- **Dyadic harmony:** Two notes imply the full chord; no stacked voicings.

## Rehearsal letter structure

- **A** — Main theme
- **B** — Development
- **C** — Thematic return
- **D** — Outro

## Performance suggestions

- **Tempo:** 88 bpm; lyrical flow.
- **Protect the tune:** Strings may carry melody; guitar provides counterpoint.
- **Warmth:** Wyble lines should feel singable, not mechanical.
