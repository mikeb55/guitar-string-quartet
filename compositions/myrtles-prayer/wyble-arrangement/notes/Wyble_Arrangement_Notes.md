# Wyble Arrangement Notes — Myrtle's Prayer

## Source file used

`compositions/myrtles-prayer/musicxml/V5_Myrtles_Prayer_Final.musicxml`

## Reason for creating the arrangement

To translate the counterpoint/tonality hybrid into Wyble two-line language. The piece already has counterpoint; the Wyble arrangement emphasises guitar as a second contrapuntal engine alongside the strings.

## Overview of the contrapuntal guitar approach

- **Upper voice:** 3rd, 7th, 9th of Dm9 (F, C, E).
- **Lower voice:** Root and 5th (D, A); lyric gravity.
- **Chamber counterpoint:** Guitar and strings share independent voice logic.
- **84 bpm:** Measured pacing.

## Rehearsal letter structure

- **A** — Intro / main theme
- **B** — Development
- **C** — Thematic return
- **D** — Coda

## Performance suggestions

- **Lyric gravity:** Two lines should feel like prayer; avoid groove.
- **Strings:** Counterpoint; melody; harmonic support.
- **Balance:** Guitar and strings as equal partners in counterpoint.
