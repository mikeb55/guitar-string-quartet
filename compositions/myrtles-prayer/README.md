# Myrtle's Prayer

Counterpoint / Tonality Hybrid + Wayne Shorter Narrative (light). Reflective, intimate, prayer-like.

## Key

D minor tonal centre; modal colour (Dm9, Bbmaj7#11, Gm9, C13sus, Fmaj9)

## Tempo

q = 84 (slow–medium pulse)

## Form

A (quiet emergence) – B (contrapuntal weaving) – C (harmonic bloom, climax) – D (suspended stillness) – A' (transformed return) – Coda (release)

## Instrumentation

Guitar, Violin I, Violin II, Viola, Cello

## Engine

Counterpoint / Tonality Hybrid (primary); Wayne Shorter Narrative (secondary, light)
