# Wyble Arrangement Notes — Myrtle's Prayer

## Source file used

`compositions/myrtles-prayer/musicxml/V5_Myrtles_Prayer_Final.musicxml`

## Purpose of the arrangement

This is an **alternate performance arrangement** built around Jimmy Wyble–style two-line guitar counterpoint. It is not a replacement for the main album version. The arrangement translates the piece into Wyble language while preserving its lyrical, intimate, spacious, and chamber-like identity.

## Where A, B, and C occur

- **Reh. A (Intro)** — Measure 1. Prayerful, intimate opening. Guitar two-line counterpoint with light string support. Prepares main material. Bars 1–16.

- **Reh. B (Optional solo / development area)** — Measure 17. Gm9/Dm9 harmonic loop. Suitable for improvisation or free extension. Guitar alternates between Wyble counterpoint and single-line improvisation. Lyric gravity preserved. Bars 17–32.

- **Reh. C (Outro)** — Measure 33. Thinning texture. Guitar contrary-motion close. Strings reduce to sustained colour. Haunting, unresolved character. Bars 33 to end.

## Modular use

The arrangement may be used **modularly**. Sections A, B, and C are musically functional on their own. In performance you may:

- Play A → B → C straight through
- Extend B for improvisation
- Use A as a standalone intro
- Use C as a standalone outro
- Repeat B as a vamp

## Performance suggestions

- **Lyric gravity:** Two lines should feel like quiet inner conversation, not jazzy or busy.
- **Strings:** Breathe around the guitar rather than accompany heavily.
- **Dynamics:** pp opening; build sparingly; return to pp for the close.
- **Tempo:** 84 bpm; measured, prayerful pacing.
