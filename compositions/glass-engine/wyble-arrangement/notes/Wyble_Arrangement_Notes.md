# Wyble Arrangement Notes — Glass Engine

## Source file used

`compositions/glass-engine/musicxml/V9_Glass_Engine_Atmospheric_Master.musicxml`

## Reason for creating the arrangement

To translate the Frisell atmosphere into Wyble two-line language. Space and restraint are preserved; guitar becomes sparse contrapuntal dyads rather than triadic comping.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Sparse; 3rd, 7th of Dm (F, C).
- **Lower voice:** Root and 5th (D, A); minimal activity.
- **Oblique motion:** One voice holds while the other moves; maximum space.
- **66 bpm:** Very slow; breathable.

## Rehearsal letter structure

- **A** — Intro / main theme
- **B** — Development
- **C** — Thematic return
- **D** — Outro

## Performance suggestions

- **Space:** Allow long rests; avoid filling.
- **pp dynamics:** Wyble lines should be whisper-quiet.
- **Strings:** Sustained pads; harmonic halo.
