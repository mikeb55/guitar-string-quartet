# working-title-01

A contemplative chamber piece in D Lydian with modal shifts to B Dorian. Guitar and strings in intimate dialogue; sparse texture, asymmetrical phrasing, space and silence as structural elements.

## Key

D Lydian (centres: D, B Dorian)

## Tempo

Andante, q = 60 (poco rubato)

## Form

A (7 bars) — B (9 bars) — A' (5 bars)

## Instrumentation

Guitar, Violin I, Violin II, Viola, Cello
