# Wyble Arrangement Notes — Sylva Narrative

## Source file used

`compositions/sylva-narrative/musicxml/V7_Sylva_Narrative_New_Intro.musicxml`

## Reason for creating the arrangement

To translate the Wayne Shorter narrative material into Jimmy Wyble's two-line contrapuntal guitar language. Preserves motivic storytelling and sectional logic while reimagining the guitar as independent melodic voices.

## Overview of the contrapuntal guitar approach

- **Upper voice:** Carries 3rd, 7th, 9th of Amaj7 (C#, G#, B) in melodic line.
- **Lower voice:** Root and 5th (A, E); bass contour.
- **Dyadic harmony:** Two-note combinations imply the full chord.
- **5/4 time:** Preserved from source; narrative pacing.

## Rehearsal letter structure

- **A** — Intro / main theme (Amaj7)
- **B** — Development / solo section
- **C** — Thematic return
- **D** — Outro

## Performance suggestions

- **Tempo:** 72 bpm; allow narrative breathing.
- **Strings:** Sustained harmonic field; avoid crowding guitar register.
- **Phrasing:** Two lines should feel like independent characters in the story.
