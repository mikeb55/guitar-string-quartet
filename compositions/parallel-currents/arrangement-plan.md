# Parallel Currents – Arrangement Plan

## Instrument Roles

| Instrument | Role |
|------------|------|
| Guitar | Two independent voices; upper melody + lower counterline |
| Violin I | Takes upper guitar voice |
| Viola | Takes lower guitar counterline |
| Violin II | Shadow voice, inversions |
| Cello | Extended bass contour |

## Texture Map

- Solo guitar two-voice
- Guitar + violin duo
- Trio counterpoint
- Full quintet weaving
- Return to two-voice
- Refresh every 8–12 measures

## Ending Logic

Gradual return to two voices. Guitar upper harmonic, lower stepwise. Ensemble fades before last interval. No tonal finality.
