# Launcher Guide

## Quick Start

**Double-click `launch-gsq-engine.hta`** to open the graphical launcher.

Or **double-click `launch-gsq-engine.bat`** if you prefer.

## Desktop Shortcut

**Double-click `create-desktop-shortcut.vbs`** once (from the repo root) to create the desktop icon.

The desktop shortcut name is: **Launch Guitar-String-Quartet Engine**

## What the Launcher Does

The launcher is a front-end for the master composition engine at:

`prompts\master-composition-engine.md`

Use the buttons to open the engine prompt, create request templates, browse compositions, and access rules, prompts, references, and scratch.

## Requirements

This version requires **no Python installation**. It uses native Windows HTA (HTML Application) and runs on standard Windows systems with no external installs.
