# Quartet Texture Rules

Reusable texture models for guitar + string quartet.

## Dark Trio Subset

**Instruments:** Guitar, Viola, Cello  
**Use when:** Intimate, low-register, contemplative  
**Characteristics:** Guitar in mid-low register; viola and cello in lower positions; no violins  
**Avoid:** Dense chord voicings; busy motion

## Split Register Bloom

**Instruments:** All five, but in distinct registers  
**Use when:** Full ensemble with transparency  
**Characteristics:** Violins high; viola mid; cello low; guitar in its own register; wide spacing  
**Avoid:** Crowding; tutti in same octave

## Inner Voice Weave

**Instruments:** Violin II, Viola, optionally Guitar  
**Use when:** Harmonic development without melody on top  
**Characteristics:** Contrapuntal interior lines; no clear lead  
**Avoid:** Mechanical parallel motion

## Cello Pedal Foundation

**Instruments:** Cello (pedal) + 1–3 others  
**Use when:** Harmonic stasis; slow change over static bass  
**Characteristics:** Cello sustains or repeats; others move above  
**Avoid:** Pedal for entire piece; no variation

## Violin Shimmer Layer

**Instruments:** Violin I and/or II, high register  
**Use when:** Ethereal, floating texture  
**Characteristics:** Sustained or tremolo; soft dynamics; sparse  
**Avoid:** Constant shimmer; generic pad

## Guitar + Viola Duet

**Instruments:** Guitar, Viola  
**Use when:** Intimate dialogue; reduced texture  
**Characteristics:** Conversational; complementary registers  
**Avoid:** Guitar dominating; viola as mere accompaniment

## Guitar + Cello Counterpoint

**Instruments:** Guitar, Cello  
**Use when:** Two-voice development; bass and melody  
**Characteristics:** Independent lines; contrapuntal  
**Avoid:** Cello as simple bass line; guitar as block chords

## When to Use Each

| Texture | Typical placement |
|---------|-------------------|
| Dark trio | Intro, bridge, sparse sections |
| Split register bloom | Climax, full ensemble moments |
| Inner voice weave | Transition, development |
| Cello pedal | Harmonic stasis, verse |
| Violin shimmer | Ethereal passages, codas |
| Guitar + viola | Intimate verses, interludes |
| Guitar + cello | Exposed duet, development |
