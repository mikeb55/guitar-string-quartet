# Engraving Rules

Expectations for MusicXML output compatible with Sibelius.

## Instrument Names

- **Guitar** — "Guitar"
- **Violin I** — "Violin I"
- **Violin II** — "Violin II"
- **Viola** — "Viola"
- **Cello** — "Cello"

Use full names; avoid abbreviations in score.

## Tempo Marks

- At start of piece
- At every tempo change
- Use standard Italian or metronome marks (e.g. "Andante", "q = 72")

## Rehearsal Letters

- At section boundaries (A, B, C, D, etc.)
- Align with form (verse, chorus, bridge)
- Visible and consistent

## Chord Symbols

- When relevant for guitar and rehearsal
- Above staff when used
- Standard notation (e.g. Cmaj7, Dm7, G7sus4)

## Readable Spacing

- Adequate measures per system (typically 4–6)
- Avoid overcrowding
- Page breaks at logical points (e.g. section boundaries)

## Clean Rhythmic Notation

- No overlapping notes
- Clear beams (eighth notes, sixteenths)
- Rests properly placed
- Dotted rhythms clear

## Sibelius Compatibility

- MusicXML 3.0 or 4.0
- Standard elements only
- Test import before finalising
