# Dual Engine Composition Mode

## Purpose

Allow compositions to be generated using two compositional engines simultaneously.

## Engine Roles

**Primary engine defines:**

- structural logic
- melodic development

**Secondary engine defines:**

- harmonic vocabulary
- texture design

## Example Pairings

- Shorter + Frisell
- Shorter + Slonimsky
- Frisell + Counterpoint
- Tonality Vault + Counterpoint
- Hill + Slonimsky

## Generation Logic

1. Generate motif using primary engine rules.
2. Generate harmonic vocabulary using secondary engine rules.
3. Combine both into a unified structural design.

## Validation

Ensure both engines influence the final composition.

The piece must show:

- motif evolution from primary engine
- harmonic vocabulary from secondary engine

If either engine influence is weak, regenerate.
