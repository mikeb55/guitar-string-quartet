# SCORE READABILITY STANDARD

All generated scores must follow professional rehearsal readability standards.

These rules apply to every composition exported as MusicXML.

---

## 1. CHORD SYMBOLS (MANDATORY)

The guitar staff must include chord symbols above the staff.

**Requirements:**

- chord symbols must reflect the actual harmony
- chord symbols must not contradict the notes
- chord symbols must appear whenever harmony changes
- avoid unnecessary repetition if harmony remains the same
- chord symbols must appear above the guitar staff in standard lead-sheet format

**Example acceptable symbols:**

- Dmaj7
- G6
- A7sus4
- Bm9
- Fmaj7#11

---

## 2. BOXED SECTION LETTERS

Each structural section must be marked with a boxed rehearsal letter.

Use capital letters:

**A** **B** **C** **D** **E** **F**

These must appear:

- above the system
- boxed
- aligned with the first measure of each new section

Sections typically include:

- motif introduction
- development
- contrast
- climax
- return
- coda

Do not create sections shorter than 6 bars unless musically necessary.

---

## 3. REHEARSAL FRIENDLY LAYOUT

Include additional rehearsal cues:

- double barlines at major structural transitions
- tempo marking at beginning
- descriptive tempo text if relevant (e.g. "slowly, floating")
- dynamic markings at major texture changes

---

## 4. CLEAR HARMONIC NAVIGATION

If the harmony changes rapidly:

- place chord symbols above each measure
- avoid clutter but ensure harmonic navigation remains clear

If harmony stays static:

- place chord symbol once and extend with continuation dash.

---

## 5. ENSEMBLE HITS

When the ensemble strikes together:

- place accent articulation
- optionally include text such as "tutti hit" or "ensemble accent"

---

## 6. SECTION TRANSITION MARKERS

At major structural transitions include one of:

- double barline
- rehearsal letter
- text indication (e.g. "build", "sparser", "climax")

---

## 7. PART CLARITY

Ensure each instrument remains readable:

| Instrument | Role |
|------------|------|
| Violin I | primary melodic line |
| Violin II | harmonic reinforcement / counterline |
| Viola | interior motion / counterpoint |
| Cello | pedal / grounding |
| Guitar | harmonic engine |

**Avoid:**

- long empty passages in inner strings
- unnecessary staff clutter
- unclear phrase boundaries
