# Repetition Fatigue Rule

Generative systems tend to over-loop material.

## Maximum Repetition

**Harmonic pattern:** max 4 bars before variation.

**Rhythmic cell:** max 3 repeats.

**Motif:** must evolve after 2 iterations.

**Texture:** must refresh every 10–16 bars.

## Required Variation

If a passage exceeds these limits it must introduce one of:

- harmonic modulation
- orchestration change
- melodic inversion
- dynamic contrast
- silence / ensemble hit
- register shift
