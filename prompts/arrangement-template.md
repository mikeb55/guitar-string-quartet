# Arrangement Template

Use when planning orchestration for a composition.

**Guitar-first rule applies:** Plan guitar harmonic material first; orchestrate strings in response. See `rules/guitar-first-composition-rule.md`.

**Anti-monotony rule applies:** Plan texture changes every 6 bars; motifs evolve; at least three structural phases. See `rules/anti-monotony-composition-rule.md`.

**Score readability standard applies:** Plan for chord symbols, boxed rehearsal letters (A, B, C…), and double barlines at section boundaries. See `rules/score-readability-standard.md`.
